# gene-expression-disease-analysis
Python-based bioinformatics project for analyzing gene expression and identifying disease-associated genes.

# Overview
This project uses Python to analyze gene expression data and identify genes that behave differently in healthy and diseased samples.

# Objectives
- Load gene expression data.
- Compare healthy and disease samples.
- Identify differentially expressed genes.
- Visualize the results.

# Tools
- Python
- Pandas
- NumPy
- Matplotlib
- SciPy
